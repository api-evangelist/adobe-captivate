Adobe Captivate 13.1’s new Classic file import feature falls short for real-world course migration. Imported projects remain heavily restricted, with limited editing capabilities, missing interactions, and persistent publishing issues. For many long-time Captivate users, the feature feels more like a dead-end workaround than a true modernization path.
