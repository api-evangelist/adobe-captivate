A user reports a display issue with the Tabs Widget in Adobe Captivate 13.1.0.197. The widget renders correctly in Captivate's editor and mobile horizontal preview, but appears distorted in browser review, with text cramped to the side and images consuming excessive space. Adobe Support suggested updating to version 13.1.1, though the user confirmed the problem persisted after the update.
